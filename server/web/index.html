API Notes

The client should use GET /api/v1/search, send the chosen resultId to POST /api/v1/downloads, then poll GET /api/v1/downloads/{gid}.

This page intentionally uses the same route contract as the Apple TV app. It does not depend on the old unversioned endpoints.
